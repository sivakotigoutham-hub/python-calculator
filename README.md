# 🧮 Advanced Python Calculator

A feature-rich, professional calculator application built with Python and Tkinter. This calculator combines a clean modern interface with powerful functionality including memory operations, history tracking, and full keyboard support.

## ✨ Features

### Basic Operations
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Decimal support
- 🔄 Parentheses support

### Advanced Features
- 🧠 Memory functions (MC, MR, M+, M-)
- 📊 Percentage calculations
- √ Square root
- x² Square
- ± Sign change
- ⌫ Backspace
- 📜 Calculation history (last 10 entries)

### User Experience
- 🎨 Modern dark theme design
- ⌨️ Full keyboard support
- 🖱️ Mouse-friendly interface
- 📱 Responsive button layout
- ⚡ Real-time expression display
